Spiral Matrix

Problem Statement

Print the given matrix in spiral format starting from the leftmost bottom element.

Input Format

The first line of the input indicates number of rows and columns in the matrix.

Output Format

Print the Matrix in Spiral Format starting from the leftmost bottom element.

Sample Input

1 2 3 4
5 6 7 8
9 3 2 1
5 4 3 2

Sample Output
